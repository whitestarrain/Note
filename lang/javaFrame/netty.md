# 概述

Netty是一个高性能的异步事件驱动网络应用框架，用于快速开发可维护的高性能协议服务器和客户端。

# Netty作用

Netty简化了TCP/UDP等网络编程，广泛用于RPC框架、消息中间件、游戏服务器等场景。

# Netty架构

Netty采用Reactor线程模型，核心组件包括Channel、EventLoop、ChannelPipeline和ChannelHandler。
