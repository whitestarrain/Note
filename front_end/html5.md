# 1. Canvas

Canvas 是 HTML5 提供的用于在网页上绘制图形的元素，通过 JavaScript 的 Canvas API 可以实现 2D/3D 图形绘制、动画、游戏等功能。

# 2. WebSockets

WebSocket 是 HTML5 提供的在单个 TCP 连接上进行全双工通信的协议，适用于需要服务器主动推送数据的实时应用场景。

# 3. Fetch API

Fetch API 是用于发起网络请求的现代接口，基于 Promise 设计，用来替代传统的 XMLHttpRequest。

# 4. File API

File API 允许 Web 应用访问用户本地文件系统中的文件内容，常配合 `<input type="file">` 或拖拽操作使用。

# 5. XMLHttpRequest(新版本)

XMLHttpRequest Level 2 在原有基础上增加了跨域请求、上传进度事件、发送二进制数据等能力。
